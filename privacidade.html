Política de Privacidade

RGPD Última atualização: 23 de abril de 2025

A presente Política de Privacidade explica como a WebWave Studio recolhe, utiliza e protege os dados pessoais dos utilizadores, em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD).

1. Dados Pessoais Recolhidos

Podemos recolher os seguintes dados pessoais:

  • Nome
  • Endereço de e-mail
  • Número de telefone
  • Informações fornecidas voluntariamente através de formulários ou contacto direto

Também podemos recolher dados técnicos de navegação (ex.: endereço IP, tipo de dispositivo, navegador e páginas visitadas), quando aplicável.

2. Finalidade do Tratamento

Os dados recolhidos podem ser utilizados para:

  • Responder a pedidos de contacto e orçamento
  • Fornecer informações sobre serviços
  • Melhorar a experiência do utilizador e a performance do website
  • Comunicação comercial, quando autorizada
  • Cumprimento de obrigações legais

3. Partilha de Dados

A WebWave Studio não vende nem cede dados pessoais a terceiros.

Os dados poderão ser partilhados apenas quando necessário para:

  • Cumprimento de obrigações legais
  • Prestadores de serviços técnicos (ex.: alojamento, e-mail), estritamente para funcionamento do website

4. Cookies

Este website pode utilizar cookies para fins funcionais e analíticos. O utilizador pode configurar ou desativar cookies no seu navegador, podendo isso afetar a experiência de navegação.

5. Direitos do Titular dos Dados

O utilizador tem o direito de:

  • Aceder aos seus dados
  • Corrigir ou atualizar dados
  • Solicitar a eliminação dos dados
  • Limitar ou opor-se ao tratamento
  • Solicitar portabilidade dos dados

Para exercer estes direitos, contacte-nos através dos contactos indicados no final desta página.

6. Segurança

Implementamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais. No entanto, nenhum sistema é totalmente seguro.

7. Conservação dos Dados

Os dados pessoais serão conservados apenas pelo período necessário às finalidades descritas, ou conforme exigido por lei.

8. Menores

Este website não se destina a menores de 13 anos e não recolhe intencionalmente dados pessoais de menores.

9. Alterações a esta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento. A versão mais recente estará sempre disponível neste website.
